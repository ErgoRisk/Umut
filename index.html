PM Case Study · Workwize · Umut Gunduz
Four features I'd ship to make Workwize
the default for global IT teams.
I spent time going through Workwize's customer reviews, mapped the competitive landscape, and looked for the gaps the whole category has ignored. These four features come directly from that research. Each one is scoped, metricked, and sequenced as a 3-month roadmap.
Feature Prototypes
Feature 1 · 3PL App
Partner portal + Ops dashboard
Replaces email and WhatsApp with a structured order queue. Partners capture serial numbers directly. Workwize ops get real-time partner health.
↗
Feature 2 · BambooHR Integration
Zero-touch onboarding
New hire webhook triggers auto-provisioning. Role, location, and start date drive equipment selection with no IT involvement.
↗
Feature 3 · Slack Integration
Approvals, lookups, self-service in Slack
Approve requests in 15 seconds with full context. Check any employee's inventory with a slash command. No tab switching.
↗
Feature 4 · Aggregated Dashboard
Global fleet visibility
Cross-region fleet health at a glance. Regional breakdown, trend charts, and one-click CSV and PDF exports for leadership.
↗
